# 🏨 Vacation Hotel Booking Website

A modern and responsive hotel booking landing page that showcases vacation destinations with attractive images, descriptions, and booking options. The project focuses on creating a clean and user-friendly interface using front-end web technologies.

---

## 📌 Features

- 🏨 Beautiful hotel and vacation destination cards
- 📱 Fully responsive design
- 🎨 Modern and clean user interface
- 🖼️ High-quality destination images
- 🛎️ "Order Now" booking buttons
- ⚡ Fast and lightweight website

---

- CSS (transforms)
